# Heart Disease Prediction 🫀

## 📌 Objective
Predict whether a person is at risk of heart disease based on health data.

## 📁 Dataset
- Heart Disease UCI Dataset (Kaggle)
- 1025 patients, 14 features

## 🤖 Model Used
- Logistic Regression (scikit-learn)

## 📊 Key Results
- Accuracy: 79.51%
- AUC Score: 0.88
- Disease Recall: 87%

## 🔍 Key Findings
- Chest pain type is the strongest predictor
- Higher max heart rate is linked to heart disease
- Model successfully detects 87% of real disease cases

## 🛠️ Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
